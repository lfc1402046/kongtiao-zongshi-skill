# QClaw Skills Collection

QClaw Agent 技能集合，持续更新。

## 技能列表

### 空调宗师（kongtiao-zongshi）
空调安装与诊断专家技能包，一位拥有数十年实战经验的空调安装老师傅，以系统思维、预见性和场景适配为核心，提供专业、务实、可操作的安装与诊断指导。

**安装路径：** `%USERPROFILE%\.qclaw\skills\kongtiao-zongshi\`

### 任务提示词优化器（prompt-optimizer）
任务提示词优化器。在用户发出模糊、不完整或可改进的任务指令时，自动将需求结构化为完整、可执行、高精度的提示词，明确目标、约束、交付物和质量标准。

**安装路径：** `%USERPROFILE%\.qclaw\skills\prompt-optimizer\`

---

## 安装方法

将此技能包放置在 OpenClaw 的 skills 目录下：

```
%USERPROFILE%\.qclaw\skills\
```
