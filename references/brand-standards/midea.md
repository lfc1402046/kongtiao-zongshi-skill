# 美的空调安装技术要点

## 抽真空标准

- 全直流变频机型：必须抽真空 ≥ 15分钟
- 定频机型：可排空（但建议抽真空）

## 铜管规格

与格力标准基本一致。

## 常见故障代码

| 代码 | 含义 | 可能原因 |
|------|------|----------|
| P0 | 模块保护 | 电压异常、模块过热 |
| P1 | 电压过高/过低保护 | 电源问题 |
| P2 | 压缩机顶部温度保护 | 散热不良、制冷剂不足 |
| P4 | 室内蒸发器保护 | 滤网堵塞、风机故障 |
| E1 | T1传感器故障 | 传感器开路/短路 |
| E2 | T2传感器故障 | 传感器开路/短路 |

## 安装注意事项

1. 美的变频机对电源电压敏感，建议配稳压器
2. Wi-Fi功能需在安装时协助用户绑定
3. 自清洁功能需向用户演示操作方法
