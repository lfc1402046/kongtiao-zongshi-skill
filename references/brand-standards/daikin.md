# 大金空调安装技术要点

## 抽真空标准

- 所有机型必须抽真空，严禁排空
- 抽真空时间：≥ 20分钟（大金标准更严格）
- 真空度：≤ -0.098MPa

## 铜管规格

| 机型 | 气管 | 液管 |
|------|------|------|
| 3HP以下 | Φ9.52/Φ12.7 | Φ6.35 |
| 4-5HP | Φ15.88 | Φ9.52 |
| 6HP以上 | Φ19.05 | Φ9.52 |

## 常见故障代码

| 代码 | 含义 | 可能原因 |
|------|------|----------|
| A0 | 外部保护装置动作 | 电源异常、外部保护触发 |
| A1 | 室内机PC板不良 | 主板故障 |
| A3 | 排水水位控制系统故障 | 排水泵故障、排水管堵塞 |
| A6 | 室内风扇电机过载/锁住 | 风机故障、电容损坏 |
| C4 | 热交换器温度传感器故障 | 传感器故障 |
| C9 | 吸入空气温度传感器故障 | 传感器故障 |
| E0 | 外部保护装置动作 | 压缩机保护 |
| E3 | 高压压力开关动作 | 散热不良、制冷剂过多 |
| E4 | 低压传感器动作 | 制冷剂泄漏 |

## 安装注意事项

1. 大金对安装工艺要求最严格，抽真空必须达标
2. VRV多联机系统需专业团队安装
3. 排水泵机型需检查排水泵工作状态
4. 建议使用大金原厂配件（铜管、保温材料）
